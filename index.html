The links above navigate to several examples of D3.js visualizations generated from data about board games. Brief descriptions of the visualizations are given below. The data for these graphs comes from Board Game Geek and includes information on games' popularity, ratings, and additional metadata

  • Chloropleth: Chloropleth Map of board game ratings by country. User can select game and hover mouse over specific country for more detailed information in data tip.
  • Force-Directed Graph: Contains an interactive force-directed graph based on a undirected graph dataset of the boardgames. Data represents the similarity between board games based on categories and game mechanics, where the edge value between board games is the weighted similarity index. Node radius is scaled based on node degree. Nodes can be dragged and pinned to allow the user to organize the graph. Pinned nodes are colored red and can be unpinned by double-clicking.
  • Interactive Line Plot: Line chart summarizing the count of board games by rating for each year from 2015 - 2019. When the user hovers over a specific rating and year, a bar chart displaying the ratings for the top 5 corresponding board games will appear below the line chart if ratings do not equal zero.
  • multiple line plots Line Plot: Line charts that visualize the number of ratings from November 2016 to August 2020 for eight board games. Four different implementations of the line charts are provided, which the user can see by scrolling down the webpage.
