The links above navigates to several examples of D3.js visualizations generated from data pertaining to board games. Brief descriptions of the visualizations are given below. The data for these graphs comes from link Board Game Geek and includes information on games' popularity, ratings, and additional metadata

  • Chloropleth: Blah Blah Blah
  • Force-Directed Graph: Contains an interactive force-directed graph based on a undirected graph dataset of the boardgames. Data represents similarity between board games based on categories and game mechanics where edge value between boardgames is the weighted similarity index. Node radius is scaled based on node degree. Nodes can be dragged and pinned to allow the user to organize the graph. Pinned nodes are colored red and can be unpinned by double clicking.
  • Interactive Line Plot: Line chart summarizing the count of board games by rating for each year from 2015 - 2019. When the user hovers over a specific rating and year, a bar chart displaying the ratings for the top 5 corresponding board games will appear below the line chart if ratings does not equal zero.
  • multiple line plots Line Plot: Line charts that visualize the number of ratings from November 2016 to August 2020 for eight boardgames. Four different imploemetations of the line charts are provided which the user can see by scrolling down the webpage.
